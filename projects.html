Projects

Some of the projects I've worked on for
research, classes, or just for fun.

ARM CPU with value prediction

We design and implement an out-of-order ARM core from the ground up in SystemC and RTL. We also develp a value prediction technique and show that it increases performance at negligible cost. The project involves developing a large set of tooling that mirrors the workflow, on a smaller scale, of an industry standard chip design company.
Collaborators: Minesh Patel, Nikhil Dixit, and Alex Hsu.

LC-3 assembler and simulator 

I develop an assembler and simulator for the LC-3 ISA described in Introduction to Computing by Dr. Yale Patt and Dr. Sanjay Patel. The tools are packaged with the third edition of the textbook and are used at multiple universities and other institutions. The frontend is built using the Electron and Vue.js frameworks to support a responsive and modern GUI across platforms. Additionally, there is a rich API designed for ease of building auto-graders.
Github page

Interactive fractal path tracer

We develop a CUDA-accelerated path tracer with a free-floating camera. The path tracer interactively renders scenes with primitive shapes and fractals using the marching rays algorithm. For some scenes, the image is noise-free within a few milliseconds.
Collaborators: Nikhil Dixit

Autonomous robot car

We built an autonomous car that navigates a maze with obstacles and other robot cars and won first place in the competition. The car runs on a real-time operating system of our own design that bridges two microcontrollers, reads several sensors, and drives four motors.
Collaborators: Minesh Patel, Nikhil Garg, Nikhil Dixit
